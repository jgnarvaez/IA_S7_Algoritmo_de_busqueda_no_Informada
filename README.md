# IA_S7_Algoritmo_de_busqueda_no_Informada

El algoritmo UCS utiliza el coste del camino desde un nodo inicial hasta cualquier nodo como
criterio para expandir de cada uno de los nodos. Por tanto, este algoritmo expandirá siempre
el de menor coste.

Un viajero desea realizar un viaje de una “Ciudad A” hasta una “Ciudad B” dentro de
Colombia. El método de transporte elegido es terrestre y para tal fin, se facilita un mapa de
conexiones de vías terrestres de las principales ciudades.

Usted deberá aplicar un algoritmo de búsqueda no informada que permita a un viajero
encontrar la ruta más corta entre un par de ciudades, recorriendo la menor cantidad de
kilómetros posible. No se permitirá el uso de diccionarios de datos, se deberán definir y crear
las estructuras de datos necesarias mediante clases, siguiendo el paradigma orientado a
objetos. Además, se deberá crear un programa principal que permita interactuar con las clases
creadas y en donde se puedan realizar diferentes pruebas. Finalmente, la información de las
ciudades y sus conexiones, deberán ser almacenada en una matriz y a partir de
esta, se pondrá en marcha la lógica del programa, haciendo uso del lenguaje de programación
Python.
